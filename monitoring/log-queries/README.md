# ログ分析クエリライブラリ

Cloud Logging のログエクスプローラーで使用できる分析クエリ集です。

## 使用方法

1. Google Cloud Console で Cloud Logging を開く
2. ログエクスプローラーに移動
3. 「クエリを入力」フィールドに以下のクエリを貼り付け
4. 時間範囲を選択して実行

## クエリファイル一覧

| ファイル | 説明 |
|---------|------|
| `error-analysis.md` | エラーログ分析クエリ |
| `performance-analysis.md` | パフォーマンス分析クエリ |
| `security-analysis.md` | セキュリティログ分析クエリ |
| `user-activity.md` | ユーザーアクティビティ分析クエリ |
| `api-metrics.md` | API メトリクス分析クエリ |
