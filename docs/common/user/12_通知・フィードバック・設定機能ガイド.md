## 1. 通知機能について

### 通知って何？

アプリからあなたのスマホに届くお知らせのことです。たとえば：

- 🏋️ 「トレーニングの時間だよ！」というリマインダー
- 📅 アカウント削除の予定がある場合のお知らせ
- 💳 支払いに関するお知らせ

### 通知を受け取るための準備

1. **スマホの設定でアプリの通知を許可する**
   - iPhoneの場合：「設定」→「通知」→「このアプリ」→「通知を許可」をON
   - Androidの場合：「設定」→「アプリ」→「このアプリ」→「通知」→ 許可をON

2. **アプリ内で通知設定を行う**
   - アプリを開く
   - 設定画面に移動
   - 「通知設定」を選択

### 通知の種類と設定項目

| 設定項目 | 説明 | 例 |
|---------|------|-----|
| 通知ON/OFF | すべての通知をまとめてON/OFFできる | ONにすると通知が届く |
| トレーニングリマインダー | 運動の時間を教えてくれる | 毎日19時に「トレーニングの時間です」 |
| リマインダー時刻 | 何時に通知が届くか | 19:00、20:30など |
| リマインダー曜日 | どの曜日に通知が届くか | 月・水・金だけ、毎日など |
| 削除通知 | アカウント削除予定のお知らせ | 30日前、7日前、1日前 |
| 課金通知 | 支払いに関するお知らせ | 支払い成功、支払い失敗 |

### トレーニングリマインダーの設定方法

1. **時刻を設定する**
   - 「リマインダー時刻」をタップ
   - 通知を受け取りたい時刻を選ぶ（例：19:00）
   - 「保存」をタップ

2. **曜日を設定する**
   - 「リマインダー曜日」をタップ
   - 通知を受け取りたい曜日をタップして選ぶ
   - 複数選べる（例：月・水・金を選ぶ）
   - 「保存」をタップ

### 注意点

- 通知設定は1時間に10回まで変更できます（あまり頻繁に変えすぎないでね）
- タイムゾーン（地域の時間帯）は自動で設定されます

## 2. フィードバック機能について

### フィードバックって何？

アプリを使った感想や、見つけた問題点を開発チームに伝える機能です。

### フィードバックの種類

| 種類 | いつ送る？ | 例 |
|-----|----------|-----|
| バグ報告 | アプリがおかしな動きをしたとき | 「記録が保存されない」「画面が固まる」 |
| 機能リクエスト | こんな機能があったらいいなと思ったとき | 「ダークモードがほしい」「友達と記録を共有したい」 |
| 一般的なフィードバック | 感想を伝えたいとき | 「使いやすい！」「デザインが好き」 |
| その他 | 上のどれにも当てはまらないとき | 「質問があります」 |

### フィードバックの送り方

1. **フィードバック画面を開く**
   - 設定画面から「フィードバックを送る」をタップ

2. **種類を選ぶ**
   - バグ報告、機能リクエスト、一般的なフィードバック、その他から選ぶ

3. **件名を入力する**
   - 何についてのフィードバックか、短く書く（100文字まで）
   - 例：「トレーニング記録が消える問題」

4. **本文を入力する**
   - 詳しい内容を書く（1000文字まで）
   - バグ報告の場合は、どんな操作をしたときに起きたか書くと◎

5. **送信ボタンをタップ**
   - 送信完了！開発チームが確認します

### フィードバックを送るときのコツ

**バグ報告の場合：**
```
❌ 悪い例：「動かない」

✅ 良い例：
「トレーニング終了後に保存ボタンを押したら、
画面が真っ白になって、記録が保存されませんでした。
スクワットを30回やった後に起きました。」
```

### 注意点

- 1日に送れるフィードバックは10件までです
- 個人情報（住所、電話番号など）は書かないでください
- すべてのフィードバックに返信できるわけではありませんが、ちゃんと読んでいます！

## 3. 設定機能について

### 設定できる項目

アプリの動作を自分好みにカスタマイズできます。

#### 🔊 音声設定

| 設定項目 | 説明 | 設定できる範囲 |
|---------|------|--------------|
| 音声ON/OFF | カウントや音声ガイドのON/OFF | ON または OFF |
| 音量 | 音の大きさ | 0〜100（0は無音、100は最大） |
| 速度 | 音声の再生速度 | 0.5〜2.0（1.0が普通の速さ） |

**音声速度の目安：**
- 0.5：ゆっくり（聞き取りやすい）
- 1.0：普通の速さ
- 1.5：少し速い
- 2.0：とても速い

#### 🎨 表示設定

| 設定項目 | 説明 | 選べるもの |
|---------|------|----------|
| テーマ | 画面の見た目 | ライト（明るい）、ダーク（暗い）、自動 |
| カメラガイド表示 | トレーニング中にガイドを表示するか | ON または OFF |
| プログレスバー表示 | 進捗を表示するか | ON または OFF |

**テーマの説明：**
- **ライト**：白を基調とした明るいデザイン（昼間におすすめ）
- **ダーク**：黒を基調としたデザイン（夜や暗い部屋でおすすめ）
- **自動**：スマホの設定に合わせて自動で切り替わる

#### 🔒 プライバシー設定

| 設定項目 | 説明 | デフォルト |
|---------|------|----------|
| 分析用データ共有 | アプリの改善のためにデータを使っていいか | OFF |
| ML用データ共有 | AI学習のためにデータを使っていいか | OFF |

**プライバシー設定について：**
- どちらも最初はOFFになっています
- ONにすると、あなたのデータ（個人を特定できない形）がアプリの改善に使われます
- 嫌なら、ずっとOFFのままで大丈夫です

### 設定の変更方法

1. アプリを開く
2. 画面右下の「設定」アイコンをタップ
3. 変更したい項目をタップ
4. 新しい値を選んで保存

### 注意点

- 設定は1時間に50回まで変更できます
- 設定はクラウドに保存されるので、別のスマホでログインしても同じ設定が使えます

## 4. よくある質問

### 通知について

**Q: 通知が届かないんですが…**

A: 以下を確認してみてください：
1. スマホの設定でアプリの通知が許可されているか
2. アプリ内の通知設定がONになっているか
3. リマインダーの曜日が正しく設定されているか
4. インターネットに接続されているか

**Q: 通知の時刻がずれるんですが…**

A: タイムゾーン（地域の時間帯）の設定を確認してください。アプリは自動的にあなたの地域を検出しますが、スマホの時刻設定が間違っていると通知もずれることがあります。

**Q: 削除通知って何ですか？**

A: アカウントの削除を予約した場合、削除の30日前、7日前、1日前に確認の通知が届きます。もし削除をキャンセルしたい場合は、設定画面から取り消せます。

---

### フィードバックについて

**Q: 送ったフィードバックに返信はありますか？**

A: 基本的に個別の返信はしていませんが、すべてのフィードバックは開発チームが確認しています。バグ報告は修正の参考に、機能リクエストは今後のアップデートの参考にさせていただきます。

**Q: 間違えてフィードバックを送ってしまいました**

A: 送信後のフィードバックは取り消せません。でも心配しないでください。間違いだとわかれば対応しません。

**Q: 「1日の上限に達しました」と表示されます**

A: フィードバックは1日10件までです。明日になるとまた送れるようになります。

---

### 設定について

**Q: 設定が保存されないんですが…**

A: 以下を確認してみてください：
1. インターネットに接続されているか
2. 設定値が正しい範囲内か（音量は0〜100、速度は0.5〜2.0など）
3. 1時間に50回以上変更していないか

**Q: ダークモードにしたのに変わらない**

A: テーマを「ダーク」に設定したあと、アプリを再起動してみてください。または「自動」を選んで、スマホの設定でダークモードをONにしてください。

**Q: 機種変更したら設定はどうなりますか？**

A: 設定はクラウドに保存されているので、同じアカウントでログインすれば自動的に引き継がれます。
