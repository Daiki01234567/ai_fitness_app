## GDPRとは？

**GDPR**は「General Data Protection Regulation」の略で、日本語では「EU一般データ保護規則」と呼ばれます。

### 簡単に言うと

あなたの個人情報（名前、メールアドレス、トレーニング記録など）を、企業やサービスがどのように使うかについて、**あなた自身がコントロールできる権利**を守るためのルールです。

### なぜ大切？

- あなたのデータは**あなたのもの**です
- 自分のデータが**どう使われているか知る権利**があります
- いつでも**データの削除を要求**できます
- 自分のデータを**持ち出す（エクスポート）**こともできます

## あなたのデータの権利

このアプリでは、以下の権利が保証されています：

| 権利 | 説明 | アプリでの操作 |
|------|------|---------------|
| **アクセス権** | 自分のデータを見る権利 | 「設定」→「データ管理」 |
| **削除権** | データを消してもらう権利 | 「設定」→「アカウント削除」 |
| **持ち運び権** | データをダウンロードする権利 | 「設定」→「データエクスポート」 |
| **同意撤回権** | 同意をやめる権利 | 「設定」→「プライバシー設定」 |

## 同意管理機能

### 同意とは？

アプリを使うとき、いくつかの「同意」を求められます：

1. **利用規約への同意** - アプリのルールに従うことへの同意
2. **プライバシーポリシーへの同意** - 個人情報の取り扱いへの同意
3. **マーケティング同意**（任意）- お知らせを受け取ることへの同意

### 同意履歴の確認方法

あなたがいつ何に同意したか、履歴を確認できます。

**アプリ画面での操作：**
```
ホーム → 設定（⚙️アイコン）→ プライバシー設定 → 同意履歴
```

**表示される情報：**
- 同意した日時
- 同意の種類（利用規約、プライバシーポリシーなど）
- 同意時のバージョン

### 同意の撤回（取り消し）

マーケティング同意など、任意の同意はいつでも取り消せます。

**注意：** 利用規約やプライバシーポリシーへの同意を撤回すると、アプリが使えなくなることがあります。

## データ削除リクエスト

### アカウント削除の流れ

```
┌─────────────────────────────────────────────────────────────┐
│  1. 削除リクエスト送信                                        │
│     ↓                                                        │
│  2. 30日間の猶予期間開始                                      │
│     （この間、データは読み取り専用になります）                   │
│     ↓                                                        │
│  3. 30日後、データが完全に削除されます                          │
└─────────────────────────────────────────────────────────────┘
```

### 削除リクエストの送信方法

1. アプリの「設定」を開く
2. 「アカウント」→「アカウント削除」を選択
3. 確認画面で「削除をリクエスト」をタップ
4. パスワードを入力して確認

### 30日間の猶予期間

削除リクエスト後、30日間は**セーフティネット期間**があります：

| できること | できないこと |
|-----------|------------|
| ✅ 自分のデータを見る | ❌ 新しいトレーニングを記録する |
| ✅ 過去の記録をダウンロード | ❌ プロフィールを変更する |
| ✅ 削除をキャンセルする | ❌ 新しいデータを追加する |

### 削除されるデータ

削除リクエストが実行されると、以下のデータが完全に削除されます：

- ユーザープロフィール（名前、メールアドレスなど）
- トレーニング記録（セッション、スコア）
- 同意履歴
- 設定情報

**注意：** 一度削除されたデータは**復元できません**。

## アカウント復旧

### 削除のキャンセル方法

30日以内であれば、削除リクエストをキャンセルできます。

**アプリでの操作：**
1. アプリにログイン
2. 画面上部の「削除予定」バナーをタップ
3. 「削除をキャンセル」を選択
4. 確認画面で「キャンセルする」をタップ

**キャンセル後：**
- すべての機能が元通り使えるようになります
- データは削除されずに残ります
- トレーニング記録の追加が再びできるようになります

### キャンセルできる期間

```
削除リクエスト日 ─────────────────→ 30日後（削除実行日）
        ↑                                    ↑
   この期間内ならキャンセル可能          この日以降はキャンセル不可
```

## データエクスポート

### エクスポートとは？

あなたのデータを**ダウンロード**して、手元に保存できる機能です。

### エクスポートできるデータ

| データ種類 | 含まれる情報 |
|-----------|------------|
| **プロフィール** | 名前、メールアドレス、登録日 |
| **トレーニング記録** | セッション履歴、スコア、種目 |
| **同意履歴** | 同意日時、同意種類 |
| **設定** | 通知設定、プライバシー設定 |

### エクスポートの手順

1. 「設定」→「データ管理」を開く
2. 「データエクスポート」をタップ
3. エクスポート形式を選択（JSON形式）
4. 「エクスポート開始」をタップ
5. 処理完了後、ダウンロードリンクが表示されます

**所要時間：** 通常1〜5分程度

## よくある質問

### Q1: 削除リクエスト後、すぐにデータは消えますか？

**A:** いいえ、30日間の猶予期間があります。この間は削除をキャンセルできます。

### Q2: 削除後、同じメールアドレスで再登録できますか？

**A:** はい、できます。ただし、以前のデータは復元されません。

### Q3: 同意履歴はなぜ保存されるのですか？

**A:** 法律により、いつ何に同意したかの記録を保持する義務があるためです。これはあなたの権利を守るためでもあります。

### Q4: データエクスポートの形式は選べますか？

**A:** 現在はJSON形式のみです。将来的にCSV形式も対応予定です。

### Q5: 退会後もデータは残りますか？

**A:** いいえ、30日の猶予期間後に完全に削除されます。ただし、法令で保持が義務付けられている監査ログは匿名化して保持される場合があります。
