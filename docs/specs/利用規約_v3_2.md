## 📝 v3.2での主な変更点

### プロジェクト懸念点分析に基づく重要な更新

✅ **セキュリティとデータ保護の強化**:
- データ削除リクエスト後の30日間の利用制限を明確化
- 削除猶予期間中のデータアクセス権限を詳細に規定
- データエクスポート可能期間の説明を追加

✅ **パフォーマンス要件の明示**:
- MediaPipe 30fps維持のための推奨端末スペックを追加

- 技術的要件の詳細化

✅ **データ侵害通知手順の追加**:
- GDPR第33条・34条に基づくインシデント対応プロセスを明示
- データ侵害発生時の通知方法とタイムラインを規定
- 利用者の権利と当社の義務を明確化

✅ **課金機能のエラーハンドリング強化**:
- 決済失敗時の対応フローを詳細化
- 返金・払い戻しポリシーの明確化
- Apple/Googleとの責任分界点を明示

✅ **BigQueryデータ処理の透明性向上**:
- データ分析用途と仮名化処理を明記
- 技術的制約と免責事項を追加
- GDPR準拠のデータ処理根拠を説明

✅ **将来機能の予告**:
- 無料プラン1日3回制限の将来実装を予告
- Phase 3ソーシャル機能の計画を明示
- 機能追加時の規約変更通知方法を説明

✅ **薬機法対応の再強化**:
- 「フォーム確認補助」の位置づけをさらに明確化
- 医療機器でない旨を複数箇所で強調
- App Store/Google Play審査対策の表現を追加

## 3. サービスの概要

### 3.1 サービスの目的

本サービスは、**トレーニングフォームの確認を補助するための参考情報を提供すること**を目的とします。

**重要**: 本サービスは以下のものではありません：
- 医療機器（薬機法に基づく承認を受けていません）
- 医学的アドバイスを提供するサービス
- 診断、治療、予防を目的とするサービス
- トレーニングの正確性を保証するサービス

### 3.2 提供する機能

本サービスは、以下の機能を提供します：

#### 3.2.1 フォーム確認補助機能

1. **骨格検出**: MediaPipe Poseを使用して、カメラに映った利用者の骨格を検出します。
2. **参考情報の提供**: 検出された骨格データに基づき、トレーニングフォームに関する参考情報を提供します。
3. **参考スコアの算出**: トレーニングフォームに関する参考スコアを算出します。

**重要な制限事項**:
- 参考情報は、あくまで参考値であり、正確性を保証するものではありません。
- 参考スコアは、医学的な評価ではなく、健康状態を判定するものでもありません。
- 利用者の最終判断を代替するものではありません。

#### 3.2.2 トレーニング記録機能

1. **履歴管理**: トレーニングの実施日時、種目、回数、セット数、参考スコア等を記録します。
2. **統計表示**: トレーニング履歴をグラフや表で可視化します。
3. **メモ機能**: トレーニングに関するメモを記録できます。

#### 3.2.3 設定機能

1. **プロフィール管理**: ニックネーム、生年月日、身長、体重等の情報を登録・編集できます。
2. **目標設定**: トレーニングの目標を設定できます。
3. **通知設定**: リマインダー通知等の設定ができます。

#### 3.2.4 その他の機能

**MVP期（Phase 1-2）**:
- 5種目（スクワット、プッシュアップ、プランク、ランジ、デッドリフト）のフォーム確認補助
- 日本語のみ対応

**Phase 3以降に追加予定の機能**:
- ソーシャル機能（友達、リーダーボード等）
- 多言語対応（英語等）
- 追加種目の対応

これらの機能は、事前の通知なく変更、追加、削除される場合があります。

### 3.3 サービスの位置づけ

1. **補助ツールとしての位置づけ**: 本サービスは、トレーニングフォームの確認を補助するツールです。
2. **自己責任の原則**: 本サービスが提供する参考情報に基づいてトレーニングを行う場合、利用者自身の判断と責任において行ってください。
3. **医療専門家への相談**: 健康上の不安がある場合、本サービスの利用前に医療専門家に相談してください。

### 3.4 利用上の注意事項

利用者は、本サービスを利用する前に、以下の事項を理解し、同意するものとします：

1. **医療機器でないこと**: 本サービスは、薬機法に基づく医療機器ではありません。
2. **参考情報の性質**: 本サービスが提供する情報は、参考情報であり、医学的アドバイスではありません。
3. **正確性の限界**: 参考情報は、必ずしも正確、完全、最新であることを保証するものではありません。
4. **利用者の責任**: トレーニングの実施、継続、中止等の最終判断は、利用者自身が行います。
5. **健康リスク**: 不適切なトレーニングは、怪我や健康被害をもたらす可能性があります。
6. **医療専門家への相談**: 健康上の問題がある場合、本サービスの利用前に医療専門家に相談してください。

## 4. 利用資格

### 4.1 年齢制限

1. 本サービスは、満13歳以上の方が利用できます。
2. 13歳未満の方は、本サービスを利用できません。
3. 未成年者（満20歳未満）が利用する場合、親権者または法定代理人の同意が必要です。

### 4.2 健康状態

以下に該当する方は、本サービスの利用前に医療専門家に相談してください：

1. 心臓病、高血圧、糖尿病等の疾患がある方
2. 妊娠中または妊娠の可能性がある方
3. 医師から運動制限を受けている方
4. 過去1年以内に大きな怪我や手術をした方
5. その他、健康上の不安がある方

**重要**: 本サービスは、上記の方の利用を禁止するものではありませんが、医療専門家に相談の上、自己責任でご利用ください。

### 4.3 技術的要件

本サービスを利用するには、以下の技術的要件を満たす必要があります：

#### 4.3.1 基本要件

| 項目 | 要件 |
|-----|------|
| **対応OS** | iOS 15.0以上、Android 8.0以上 |
| **インターネット接続** | Wi-FiまたはLTE/4G/5G接続 |
| **カメラ** | 前面または背面カメラ（720p以上） |
| **ストレージ** | 100MB以上の空き容量 |

#### 4.3.2 推奨端末スペック（v3.2で追加）

MediaPipe 30fpsを維持し、快適にご利用いただくため、以下のスペックを推奨します：

| 項目 | iOS | Android |
|-----|-----|---------|
| **OS** | iOS 15.0以上 | Android 8.0以上 |
| **RAM** | 2GB以上 | 3GB以上 |
| **CPU** | A9チップ以上 | Snapdragon 660以上 |
| **カメラ** | 720p以上 | 720p以上 |

**低スペック端末での利用について**:
- 推奨スペック未満の端末では、フレームレートが低下し、参考情報の精度が低下する可能性があります。
- アプリは自動的にパフォーマンスを監視し、必要に応じて解像度を下げて動作を軽くします。
- 低スペック端末での利用は、利用者の自己責任とします。

### 4.4 利用環境

1. **安全な環境**: 利用者は、安全な環境でトレーニングを行うものとします。
2. **十分なスペース**: トレーニングに十分なスペースを確保してください。
3. **適切な照明**: カメラで骨格を検出できる程度の照明を確保してください。
4. **端末の固定**: 端末を安定した場所に固定してください。

3. アカウント削除リクエスト後、30日間の猶予期間が設定されます。

#### 14.3.2 削除猶予期間中の制限（v3.2で追加）

アカウント削除リクエスト後、30日間の猶予期間中は、以下の制限が適用されます：

| 制限項目 | 内容 |
|---------|------|
| **新規データの作成** | 禁止（新しいトレーニングセッションを開始できません） |
| **既存データの変更** | 禁止（メモ、プロフィール等の変更ができません） |
| **データの読み取り** | 許可（トレーニング履歴の閲覧、データエクスポートが可能） |
| **ログイン** | 禁止（認証は無効化されます） |
| **データエクスポート** | 許可（メール等で依頼可能） |

**削除猶予期間の目的**:
- 利用者が誤ってアカウント削除をリクエストした場合の復旧期間
- GDPR第17条（消去権）に基づく30日以内の完全削除の実施
- データエクスポートの機会提供

#### 14.3.3 削除のキャンセル

1. 削除猶予期間中（30日間）であれば、削除をキャンセルできます。
2. 削除キャンセルは、当社サポート（support@example.com）にご連絡ください。
3. 削除キャンセル後、通常通りアカウントを利用できます。

#### 14.3.4 削除されるデータ

アカウント削除により、以下のデータが完全に削除されます：

| データ種別 | 削除タイミング | 復元可否 |
|-----------|--------------|---------|
| **アカウント情報** | 削除実行日から30日後 | 不可 |
| **プロフィール情報** | 削除実行日から30日後 | 不可 |
| **トレーニング履歴** | 削除実行日から30日後 | 不可 |
| **骨格座標データ** | 削除実行日から30日後 | 不可 |
| **設定情報** | 削除実行日から30日後 | 不可 |
| **仮名化データ（BigQuery）** | 削除実行日から30日後 | 不可 |

**重要**: 削除されたデータは、完全に復元できません。削除前に必要なデータをエクスポートしてください。

### 14.4 アカウント削除後の取り扱い

1. アカウント削除後も、法令に基づき保存が義務付けられているデータ（課金履歴等）は、法定期間保存されます。

| 通知方法 | 内容 |

**通知内容**:
1. データ侵害の性質の説明
2. データ保護責任者の連絡先
3. データ侵害の結果および影響の説明

### 16.3 データ侵害の分類

当社は、データ侵害を以下のように分類し、対応します：

#### 16.3.1 高リスク侵害

以下の場合、利用者への通知が必須です：

1. 氏名、メールアドレス、生年月日等の個人情報が流出した場合
2. ユーザーIDとパスワードが流出した場合
3. 決済情報が流出した場合
4. 健康情報（トレーニング履歴、身体情報等）が流出した場合

#### 16.3.2 中リスク侵害

以下の場合、監督機関への通知が必須ですが、利用者への通知は任意です：

1. 仮名化されたデータが流出した場合
2. 一部のログデータが流出した場合
3. 影響範囲が限定的な場合

#### 16.3.3 低リスク侵害

以下の場合、監督機関への通知も任意です：

1. 暗号化されたデータが流出した場合（暗号鍵は流出していない）
2. 公開されている情報のみが流出した場合

### 16.4 利用者が取るべき対策

データ侵害の通知を受けた場合、利用者は以下の対策を講じることを推奨します：

1. **パスワード変更**: 直ちにパスワードを変更してください。

3. **クレジットカード会社への連絡**: 決済情報が流出した場合、クレジットカード会社に連絡してください。
4. **監督機関への報告**: 必要に応じて、監督機関（個人情報保護委員会等）に報告してください。

### よくある質問（FAQ）

#### Q1. 本サービスは医療機器ですか？
**A**: いいえ、本サービスは薬機法に基づく医療機器ではありません。トレーニングフォームの確認を補助するための参考情報を提供するツールです。

#### Q2. 参考スコアは絶対的な評価ですか？
**A**: いいえ、参考スコアはあくまで参考値であり、絶対的な評価ではありません。医学的な評価でもありません。

#### Q3. アカウント削除後、データは復元できますか？
**A**: いいえ、アカウント削除後、すべてのデータは完全に削除され、復元できません。削除前に必要なデータをエクスポートしてください。

#### Q4. 無料プランは永久に無制限ですか？
**A**: MVP期（Phase 1-2）では無制限ですが、Phase 3以降、1日3回までの制限が導入される予定です。制限導入の30日前に通知します。

#### Q5. 決済が失敗した場合、どうなりますか？
**A**: Apple/Googleが自動的に再試行します（最大3回）。7日間決済が完了しない場合、有料プラン機能が制限されます。30日間決済が完了しない場合、無料プランに移行します。

#### Q6. 低スペック端末でも使えますか？
**A**: 推奨スペック未満の端末でも利用可能ですが、フレームレートが低下し、参考情報の精度が低下する可能性があります。自己責任でご利用ください。

#### Q7. データ侵害が発生した場合、どうなりますか？
**A**: 当社は、データ侵害を認識してから72時間以内に、監督機関と利用者に通知します。詳細は第16条を参照してください。

#### Q8. 規約はいつ変更されますか？
**A**: 当社は必要に応じて規約を変更できます。重要な変更の場合、施行日の30日前に通知します。
