# 067 マスタデータ管理

## 概要
アプリで使用するマスタデータ（種目、カテゴリ、設定値など）を管理者が編集できる機能を実装します。

## Phase
Phase 4（管理者・運用者・多言語）

## 依存チケット
- 061: 管理者ダッシュボード

## 要件
- 種目マスタ管理
- カテゴリマスタ管理
- 設定値管理
- バージョン管理
- ロールバック機能

## 受け入れ条件
- [ ] 種目の追加/編集/無効化ができる
- [ ] カテゴリの追加/編集ができる
- [ ] アプリ設定値を変更できる
- [ ] 変更履歴が記録される
- [ ] 問題があった場合にロールバックできる

## 参照ドキュメント
- `docs/expo/specs/05_要件定義書_Expo版_v1_Part5.md` - データ構造

## 技術詳細
- Firestore（master_data コレクション）
- Remote Configとの連携
- 変更履歴のバージョン管理
- キャッシュ無効化機構

## マスタデータ種別
| 種別 | 説明 | 例 |
|------|------|-----|
| exercises | 種目定義 | スクワット、プッシュアップ |
| categories | カテゴリ | 下半身、上半身、全身 |
| app_config | アプリ設定 | 最大セッション時間、スコア閾値 |
| messages | システムメッセージ | エラーメッセージ、ガイド文 |
