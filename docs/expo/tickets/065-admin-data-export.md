# 065 データエクスポート（管理者）

## 概要
管理者が分析やレポート作成のために各種データをエクスポートできる機能を実装します。

## Phase
Phase 4（管理者・運用者・多言語）

## 依存チケット
- 060: 管理者認証基盤
- 061: 管理者ダッシュボード

## 要件
- ユーザーデータエクスポート
- 利用統計エクスポート
- 売上データエクスポート
- 監査ログエクスポート
- 定期エクスポート設定

## 受け入れ条件
- [ ] 各種データをCSV/JSONでエクスポートできる
- [ ] 期間やフィルター条件を指定できる
- [ ] 大量データの場合はバックグラウンド処理される
- [ ] エクスポート完了時に通知が届く
- [ ] 定期エクスポート（日次/週次）が設定できる

## 参照ドキュメント
- `docs/expo/specs/05_要件定義書_Expo版_v1_Part5.md` - データ管理

## 技術詳細
- Cloud Functionsによるバックグラウンド処理
- Firebase Storageへの一時保存
- BigQueryからの直接エクスポート
- ダウンロードリンクの有効期限（24時間）

## エクスポート可能データ
| データ | 権限 | 形式 |
|--------|------|------|
| ユーザー一覧 | admin | CSV |
| 利用統計 | admin, operator | CSV, JSON |
| 売上レポート | admin | CSV, PDF |
| 監査ログ | admin | CSV |
