# 069 端末情報管理

## 概要
ユーザーが使用している端末情報を管理し、分析やサポートに活用する機能を実装します。

## Phase
Phase 4（管理者・運用者・多言語）

## 依存チケット
- 061: 管理者ダッシュボード

## 要件
- 端末情報の収集
- 端末一覧表示
- OS/バージョン分布
- アプリバージョン分布
- 端末別の問題分析

## 受け入れ条件
- [ ] ユーザーの端末情報（OS、バージョン、デバイス名）が収集される
- [ ] 管理画面で端末情報の分布が確認できる
- [ ] 古いOS/アプリバージョンの使用率が把握できる
- [ ] 特定端末でのクラッシュ率が分析できる
- [ ] プライバシーに配慮した匿名化がされている

## 参照ドキュメント
- `docs/expo/specs/02_要件定義書_Expo版_v1_Part2.md` - 非機能要件

## 技術詳細
- Expo Device APIによる端末情報取得
- Firestoreへの端末情報保存
- BigQueryでの集計分析
- 端末ID（匿名化済み）によるユーザー紐付け

## 収集情報
| 項目 | 例 | 用途 |
|------|-----|------|
| OS | iOS 17.2, Android 14 | 互換性分析 |
| デバイスモデル | iPhone 15, Pixel 8 | パフォーマンス分析 |
| アプリバージョン | 1.2.3 | 更新促進 |
| 画面サイズ | 390x844 | UI最適化 |
