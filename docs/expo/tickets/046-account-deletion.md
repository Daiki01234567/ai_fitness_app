# 046 アカウント削除機能

## 概要
GDPRの「忘れられる権利」に基づき、ユーザーがアカウントと全データを削除できる機能を実装します。

## Phase
Phase 2（機能実装）

## 依存チケット
- 018: GDPR基盤実装

## 要件
- gdpr_requestAccountDeletion API
- 30日間の猶予期間
- 猶予期間中のキャンセル機能
- 猶予期間中はデータ読み取り専用
- 完全削除処理

## 受け入れ条件
- [ ] 削除リクエストを送信できる
- [ ] 30日間の猶予期間が設定される
- [ ] 猶予期間中にキャンセルできる
- [ ] 猶予期間中は新規データ作成不可
- [ ] 30日後にデータが完全に削除される
