# 062 ユーザーアカウント管理

## 概要
管理者がユーザーアカウントを検索・管理できる機能を実装します。

## Phase
Phase 4（管理者・運用者・多言語）

## 依存チケット
- 060: 管理者認証基盤
- 061: 管理者ダッシュボード

## 要件
- ユーザー検索（メール、ID、名前）
- ユーザー詳細表示
- アカウント停止/有効化
- プランの手動変更
- パスワードリセット発行

## 受け入れ条件
- [ ] ユーザーをメールアドレスで検索できる
- [ ] ユーザーの詳細情報が表示される
- [ ] アカウントを停止/有効化できる
- [ ] プランを手動で変更できる（admin権限のみ）
- [ ] 操作履歴が監査ログに記録される

## 参照ドキュメント
- `docs/expo/specs/05_要件定義書_Expo版_v1_Part5.md` - ユーザーデータ管理

## 技術詳細
- Firestoreクエリによるユーザー検索
- Firebase Admin SDKによるアカウント操作
- 操作の監査ログ記録

## 操作権限
| 操作 | admin | operator | viewer |
|------|-------|----------|--------|
| 検索 | O | O | O |
| 詳細閲覧 | O | O | O |
| 停止/有効化 | O | O | X |
| プラン変更 | O | X | X |
| 削除 | O | X | X |
