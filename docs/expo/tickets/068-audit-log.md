# 068 監査ログ検索

## 概要
システム操作の監査ログを検索・閲覧できる機能を実装します。GDPR準拠のための証跡管理を行います。

## Phase
Phase 4（管理者・運用者・多言語）

## 依存チケット
- 060: 管理者認証基盤
- 061: 管理者ダッシュボード

## 要件
- ログ検索機能
- フィルタリング（日時、ユーザー、操作種別）
- ログ詳細表示
- ログエクスポート
- 改ざん検知

## 受け入れ条件
- [ ] 監査ログを日時範囲で検索できる
- [ ] ユーザーIDや操作種別でフィルタリングできる
- [ ] ログの詳細（操作者、日時、変更前後の値）が確認できる
- [ ] ログをCSVでエクスポートできる
- [ ] ログの改ざんが検知される

## 参照ドキュメント

## 技術詳細
- Firestore（audit_logs コレクション）
- 書き込み専用（更新/削除不可）
- ハッシュチェーンによる改ざん検知
- 90日保存（それ以降はCloud Storageにアーカイブ）

## 記録される操作
| カテゴリ | 操作例 |
|----------|--------|
| 認証 | ログイン、ログアウト、認証失敗 |
| ユーザー | 作成、更新、削除、停止 |
| 課金 | サブスク変更、決済 |
| 管理者 | 設定変更、エクスポート |
