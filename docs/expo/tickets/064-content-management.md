# 064 コンテンツ管理

## 概要
アプリ内のコンテンツ（お知らせ、FAQ、ヘルプ記事など）を管理者が編集できる機能を実装します。

## Phase
Phase 4（管理者・運用者・多言語）

## 依存チケット
- 061: 管理者ダッシュボード

## 要件
- お知らせ作成/編集/削除
- FAQ管理
- ヘルプ記事管理
- 公開/非公開設定
- 公開予約機能

## 受け入れ条件
- [ ] お知らせを作成・編集・削除できる
- [ ] FAQのカテゴリと質問を管理できる
- [ ] コンテンツの公開/非公開を切り替えられる
- [ ] 公開日時を予約設定できる
- [ ] プレビュー機能が動作する

## 参照ドキュメント
- `docs/expo/specs/02_要件定義書_Expo版_v1_Part2.md` - FR-030, FR-031

## 技術詳細
- Firestore（notices, faqs, help_articles）
- マークダウンエディタ
- Cloud Schedulerによる公開予約
- 画像アップロード（Firebase Storage）

## コンテンツタイプ
| タイプ | 説明 |
|--------|------|
| お知らせ | 新機能、メンテナンス情報など |
| FAQ | よくある質問と回答 |
| ヘルプ | 使い方ガイド |
| 利用規約 | 法的文書（バージョン管理） |
