# 031 トレーニング実行画面

## 概要
実際にトレーニングを行う画面を実装します。リアルタイムのフォームチェックと進捗表示を行います。

## Phase
Phase 2（機能実装）

## 依存チケット
- 022: 骨格検出基盤
- 030: カメラ設定画面

## 要件
- カメラプレビュー上に骨格をオーバーレイ表示
- リアルタイムフォーム評価表示
- 回数表示（レップカウント）
- 経過時間表示
- 進捗バー（YouTube風）
- 一時停止/再開機能
- 終了ボタン
- 体認識中断時の一時停止と自動再開

## 受け入れ条件
- [ ] 骨格がリアルタイムで表示される
- [ ] フォームの参考情報が表示される
- [ ] レップカウントが正確に動作する
- [ ] 体が認識できない時に一時停止する
- [ ] 30秒認識できない場合に確認ダイアログが表示される
- [ ] 終了ボタンで結果画面に遷移する
