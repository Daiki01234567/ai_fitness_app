# 056 無料トライアル機能

## 概要
新規ユーザーに7日間の無料トライアルを提供し、有料機能を体験できる機能を実装します。

## Phase
Phase 3（Apple認証・課金）

## 依存チケット
- 053: サブスクリプション管理

## 要件
- 7日間の無料トライアル
- トライアル開始フロー
- トライアル残日数表示
- トライアル終了通知
- トライアル→有料への自動移行

## 受け入れ条件
- [ ] 新規ユーザーが無料トライアルを開始できる
- [ ] トライアル残日数が正確に表示される
- [ ] トライアル終了3日前に通知が届く
- [ ] トライアル終了時に自動的に有料プランに移行する
- [ ] 1ユーザー1回のみトライアル利用可能

- トライアル状態のFirestore管理
- Cloud Schedulerによる終了通知
