# 025 プッシュアップ評価ロジック

## 概要
プッシュアップ（腕立て伏せ）のフォーム評価ロジックを実装します。

## Phase
Phase 2（機能実装）

## 依存チケット
- 023: フォーム評価エンジン

## 要件
- 体のライン評価（頭から足まで一直線）
- 肘角度の計測（下で約90度）
- 肘の開き具合チェック（体に対して45度程度）
- 手の位置評価（肩幅程度）
- レップカウント機能

## 受け入れ条件
- [ ] 体のラインが一直線かどうかを判定できる
- [ ] 肘角度（90度）を正確に計測できる
- [ ] お尻が上がりすぎ/お腹が下がりすぎを検出できる
- [ ] レップカウントが正確に動作する
- [ ] 0-100点のスコアリングが動作する

## 参照ドキュメント
- `docs/expo/specs/04_要件定義書_Expo版_v1_Part4.md` - フォーム評価ロジック

## 技術詳細
- MediaPipe Poseの関節データを使用
- 状態マシン: 上位置（肘160度以上）→下位置（肘100度以下）→上位置で1レップ
- 体のライン判定: 肩・腰・足首の3点が直線上にあるか確認
