# 026 アームカール評価ロジック

## 概要
アームカール（ダンベルで力こぶを鍛える運動）のフォーム評価ロジックを実装します。

## Phase
Phase 2（機能実装）

## 依存チケット
- 023: フォーム評価エンジン

## 要件
- 肘角度の計測（30度〜160度の可動域）
- 肘の固定度チェック（体の横で動かさない）
- 反動使用の検出（体を揺らさない）
- 手首角度チェック（まっすぐ維持）
- レップカウント機能

## 受け入れ条件
- [ ] 肘角度（30-160度）を正確に計測できる
- [ ] 肘が前に出ていることを検出できる
- [ ] 反動（体の揺れ）を検出できる
- [ ] レップカウントが正確に動作する
- [ ] 0-100点のスコアリングが動作する

- 反動検出: 肩の位置変化を監視
