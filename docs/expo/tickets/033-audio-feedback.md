# 033 音声フィードバック機能

## 概要
トレーニング中に音声でフォームの参考情報を伝える機能を実装します。

## Phase
Phase 2（機能実装）

## 依存チケット
- 031: トレーニング実行画面

## 要件
- expo-speechを使用した音声合成
- 日本語音声ガイダンス
- フィードバック内容
  - 「参考: 良いフォームです」
  - 「参考: 膝が前に出すぎているかもしれません」
  - 「参考: 背中をまっすぐにしてみましょう」
  - レップカウント読み上げ
- 音量調整
- 読み上げ速度調整
- ON/OFF切替

## 受け入れ条件
- [ ] 日本語で音声が再生される
- [ ] フォームの参考情報が音声で伝えられる
- [ ] レップカウントが読み上げられる
- [ ] 音量・速度が設定画面から変更できる
- [ ] ON/OFFが切り替えられる
