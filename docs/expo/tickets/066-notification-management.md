# 066 通知管理

## 概要
管理者がプッシュ通知を送信・管理できる機能を実装します。

## Phase
Phase 4（管理者・運用者・多言語）

## 依存チケット
- 061: 管理者ダッシュボード
- 043: プッシュ通知基盤

## 要件
- 通知作成・送信
- セグメント配信
- 予約配信
- 送信履歴
- 開封率分析

## 受け入れ条件
- [ ] 通知タイトル・本文を入力して送信できる
- [ ] ユーザーセグメント（全員、有料会員、非アクティブなど）を選択できる
- [ ] 配信日時を予約できる
- [ ] 過去の送信履歴が確認できる
- [ ] 開封率が集計される

## 参照ドキュメント
- `docs/expo/specs/02_要件定義書_Expo版_v1_Part2.md` - FR-020

## 技術詳細
- Firebase Cloud Messaging（FCM）
- トピックベースの配信
- Cloud Schedulerによる予約配信
- BigQueryでの開封率分析

## セグメント例
| セグメント | 条件 |
|-----------|------|
| 全員 | 全ユーザー |
| 有料会員 | subscriptionStatus = 'active' |
| 無料ユーザー | subscriptionStatus = 'free' |
| 非アクティブ | 7日以上ログインなし |
| トライアル中 | トライアル期間内 |
