# 027 サイドレイズ評価ロジック

## 概要
サイドレイズ（腕を横に上げて肩を鍛える運動）のフォーム評価ロジックを実装します。

## Phase
Phase 2（機能実装）

## 依存チケット
- 023: フォーム評価エンジン

## 要件
- 腕の挙上角度計測（70度〜90度が理想）
- 左右対称性チェック（両腕が同じ高さ）
- 体幹の安定性チェック（体を揺らさない）
- 肘の角度チェック（軽く曲げた状態を維持）
- レップカウント機能

## 受け入れ条件
- [ ] 腕の挙上角度（70-90度）を計測できる
- [ ] 左右の腕の高さの差を検出できる
- [ ] 体幹の揺れを検出できる
- [ ] レップカウントが正確に動作する
- [ ] 0-100点のスコアリングが動作する

## 技術詳細
- 状態マシン: 下位置（角度20度以下）→上位置（角度70度以上）→下位置で1レップ
- 左右対称性: 両腕の角度差が10度以内を推奨
