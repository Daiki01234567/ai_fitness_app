# 059 解約機能

## 概要
ユーザーがサブスクリプションを解約できる機能を実装します。GDPR準拠の解約フローを提供します。

## Phase
Phase 3（Apple認証・課金）

## 依存チケット
- 053: サブスクリプション管理

## 要件
- 解約リクエスト機能
- 解約理由の選択
- 解約確認ダイアログ
- 期間終了までのサービス継続
- 再開オプション

## 受け入れ条件
- [ ] 解約リクエストが送信できる
- [ ] 解約理由を選択できる（任意）
- [ ] 解約前に確認ダイアログが表示される
- [ ] 解約後も現在の請求期間終了までサービスが使える
- [ ] 解約取り消し（再開）ができる

## 技術詳細
- Stripe Subscription Cancel API
- 解約理由の分析用ログ
- 期間終了時の自動ダウングレード

## 解約理由オプション
- 使用頻度が低い
- 料金が高い
- 他のサービスを利用
- 機能に不満
- その他（自由入力）
