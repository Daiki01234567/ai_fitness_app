# 071 多言語対応基盤

## 概要
アプリの多言語対応（国際化: i18n）基盤を構築し、将来の英語対応に備えます。

## Phase
Phase 4（管理者・運用者・多言語）

## 依存チケット
- 002: 開発環境構築（Expo）

## 要件
- i18n基盤構築
- 日本語リソースの分離
- 言語切り替え機能
- 翻訳管理システム
- RTL対応の基盤

## 受け入れ条件
- [ ] 全てのUI文字列がリソースファイルに分離される
- [ ] 言語設定に応じて表示言語が切り替わる
- [ ] 端末の言語設定を自動検出できる
- [ ] 新しい言語を追加する手順がドキュメント化されている
- [ ] 日付/時刻/数値のフォーマットがロケールに対応する

## 参照ドキュメント
- `docs/expo/specs/06_要件定義書_Expo版_v1_Part6.md` - 開発計画

## 技術詳細
- react-i18next / expo-localization
- JSON形式の言語リソースファイル
- Context APIによる言語状態管理
- 動的インポートによる言語ファイル分割

## 対応言語（計画）
| 言語 | コード | 優先度 |
|------|--------|--------|
| 日本語 | ja | MVP（必須） |
| 英語 | en | Phase 4 |
| 中国語（簡体）| zh-CN | 将来 |
