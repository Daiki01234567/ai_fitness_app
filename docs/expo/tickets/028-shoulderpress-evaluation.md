# 028 ショルダープレス評価ロジック

## 概要
ショルダープレス（ダンベルを頭上に押し上げる運動）のフォーム評価ロジックを実装します。

## Phase
Phase 2（機能実装）

## 依存チケット
- 023: フォーム評価エンジン

## 要件
- 肘角度の計測（90度スタート、上で伸ばす）
- 軌道の垂直性チェック（まっすぐ上に押し上げる）
- 腰の安定性チェック（腰を反らさない）
- 左右対称性チェック（両腕が同じ高さ）
- レップカウント機能

## 受け入れ条件
- [ ] 肘角度を正確に計測できる
- [ ] 腕の軌道が垂直かどうか判定できる
- [ ] 腰の反りを検出できる
- [ ] 左右の高さの差を検出できる
- [ ] レップカウントが正確に動作する
- [ ] 0-100点のスコアリングが動作する

- 状態マシン: 下位置（肘90度以下）→上位置（肘160度以上）→下位置で1レップ
- 腰の反り検出: 肩・腰・膝のライン角度を監視
