# 063 利用状況集計

## 概要
BigQueryのデータを使用して、詳細な利用状況分析レポートを表示する機能を実装します。

## Phase
Phase 4（管理者・運用者・多言語）

## 依存チケット
- 061: 管理者ダッシュボード
- 044: BigQueryパイプライン構築

## 要件
- 日次/週次/月次レポート
- ユーザー行動分析
- 種目別利用統計
- リテンション分析
- コホート分析

## 受け入れ条件
- [ ] 日/週/月単位でレポートが生成される
- [ ] 種目別の利用回数が集計される
- [ ] 継続率（リテンション）が計算される
- [ ] データがCSVでエクスポートできる
- [ ] 期間指定でフィルタリングできる

## 参照ドキュメント
- `docs/expo/specs/04_要件定義書_Expo版_v1_Part4.md` - BigQuery設計

## 技術詳細
- BigQueryへのクエリ実行
- スケジュールクエリによる事前集計
- レポートのキャッシュ（1時間）

## 分析指標
| カテゴリ | 指標 |
|----------|------|
| ユーザー | DAU, WAU, MAU, 新規登録数 |
| エンゲージメント | セッション数, セッション時間, 種目別回数 |
| リテンション | D1, D7, D30継続率 |
| 収益 | MRR, ARPU, 解約率 |
