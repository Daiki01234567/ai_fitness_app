# 029 メニュー選択画面

## 概要
5種目のトレーニングメニューを選択する画面を実装します。

## Phase
Phase 2（機能実装）

## 依存チケット
- 015: ボトムナビゲーション

## 要件
- 5種目をカード形式で表示
  - スクワット（squat）
  - プッシュアップ（pushup）
  - アームカール（armcurl）
  - サイドレイズ（sideraise）
  - ショルダープレス（shoulderpress）
- 各種目に以下を表示
  - 種目名（日本語）
  - 難易度
  - ターゲット部位
  - 前回の記録（あれば）
- タップで種目詳細画面へ遷移

## 受け入れ条件
- [ ] 5種目がカード形式で表示される
- [ ] 各種目の情報が正しく表示される
- [ ] 前回の記録が表示される
- [ ] タップで次の画面に遷移できる
- [ ] ローディング状態が適切に表示される
