# 070 セキュリティ監視

## 概要
不正アクセスや異常な動作を検知し、管理者にアラートを送信するセキュリティ監視機能を実装します。

## Phase
Phase 4（管理者・運用者・多言語）

## 依存チケット
- 060: 管理者認証基盤
- 006: 監視・ログ基盤構築

## 要件
- 不正ログイン試行検知
- 異常なAPI呼び出し検知
- リアルタイムアラート
- セキュリティダッシュボード
- インシデント管理

## 受け入れ条件
- [ ] 連続したログイン失敗が検知される
- [ ] 異常なAPI呼び出しパターンが検知される
- [ ] 検知時に管理者にメール/Slack通知が届く
- [ ] セキュリティイベントがダッシュボードに表示される
- [ ] インシデントの対応状況を管理できる

## 技術詳細
- Cloud Monitoring / Cloud Logging
- Cloud Functionsによるアラート処理
- Firebase App Checkの異常検知
- Slack/Email連携

## 監視項目
| 項目 | 閾値 | アクション |
|------|------|-----------|
| ログイン失敗 | 5回/5分 | アカウントロック |
| API呼び出し | 1000回/分 | レート制限 |
| 不正トークン | 検知時 | セッション無効化 |
| 管理者ログイン | 全件 | ログ記録 |
